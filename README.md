# Disclosures

A concise collection of disclosure documents, analysis notes, diagrams, and
supporting data related to motor Remaining Useful Life (RUL) research and the
system architecture used in that work.

---

## Contributing

- Add new disclosure documents as markdown files in the repository root or a
  clearly named subfolder.
- If your contribution includes data, include a short README in the folder
  describing the data source, format, and any preprocessing steps.
- Submit a pull request with a brief summary of changes and any data or figure
  dependencies.

## Data & licensing

- Where applicable, data sources and license information are noted inside each
  disclosure file or its folder. If no license is provided, contact the
  repository owner before reusing datasets.
